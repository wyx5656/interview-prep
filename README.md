# Interview Preparation

面试准备代码刷题仓库，结合代码实践复习基础知识，逐步完善面试考点。

## 📋 项目目标

这个仓库用于系统性地准备技术面试，特点是：
- **代码优先**：每个知识点都有可运行的代码实现
- **详细注释**：代码中包含详细的原理说明，便于理解记忆
- **逐步完善**：按照面试考点分模块逐步补充内容

## 📁 目录结构

```
.
├── C指针核心总结.md          # C指针知识点总结笔记
├── programming-basics/      # 编程语言基础（C语言）
├── data-structures/         # 经典数据结构实现
├── algorithms/              # 算法题解和常用算法模板
├── system-design/           # 系统设计资料
├── project-review/          # 项目经验整理
└── code-templates/          # 面试常用代码模板
```

## 🎯 开发流程

1. 每次新增知识点/代码，先在 README 更新进度勾选框
2. 代码放到对应目录，添加详细注释说明原理和实现
3. 提交后推送到 GitHub，随时可以查看
4. 需要复习时可以直接看代码和注释，比纯文字记忆更深刻

## 💻 编译运行示例

```bash
# 编译运行排序算法示例
cd data-structures
gcc -o sort sort.c
./sort
```
